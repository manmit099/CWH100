Concept explanations

Programming
Programming is a way to tell computers what to do. We communicate with computers using programming languages, just like we use Hindi, English, or Spanish. Examples: Python, C, C++, Java.
Python
Dynamically typed, general‑purpose, high‑level language. Supports object‑oriented & functional paradigms. Interpreted (runs line by line). Created by Guido van Rossum (started 1989, released ~1991).
Features
Simple, platform‑independent, open‑source, huge library support. Everything in Python is an object.
Applications
Web apps, data science, machine learning, database, automation, mathematics.
Replit
Collaborative browser‑based IDE. Write & run Python without installing anything. Files/projects are called “repls”. Perfect for slow computers.
Python REPL
REPL = Read‑Evaluate‑Print Loop. Interactive shell: type python3 in terminal, get immediate results. Exit with exit(). Works as a quick calculator.
Installation
On Windows: tick Add Python to PATH. On Linux/Mac use python3 command (not python) to avoid errors.
First program
print("Hello World") is the basic output function. Prints text or numbers to the console.

[04]

Tips & Warnings

Use Replit if your computer is slow – all you need is a browser tab.
On Windows, tick the Add Python to PATH checkbox during installation.
Use python3 (not python) in the terminal on Linux/Mac to avoid errors.
Bookmark the 100 Days of Code Python playlist so you never miss a day.
The Python REPL (python3) lets you use Python as a quick calculator.
On Windows: forgetting to check Add Python to PATH means Python won't work in the terminal.
